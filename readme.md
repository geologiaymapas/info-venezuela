# 🇻🇪 Info Venezuela

Repositorio colaborativo de información geográfica y datos de interés público de Venezuela.

El objetivo de este proyecto es centralizar, organizar y mantener información estructurada sobre infraestructura y servicios esenciales del país, facilitando su uso en proyectos de análisis de datos, sistemas de información geográfica (SIG), aplicaciones web, investigación académica y proyectos de impacto social.

### Hospitales

Contiene información relacionada con centros de salud públicos y privados.

Posibles atributos:

- Nombre del hospital
- Dirección
- Estado
- Municipio
- Parroquia
- Coordenadas geográficas
- Tipo de establecimiento
- Teléfono de contacto
- Organismo responsable

### Centros de Acopio

Contiene información sobre centros de recepción, distribución y almacenamiento de ayuda humanitaria, donaciones o suministros.

Posibles atributos:

- Nombre del centro
- Dirección
- Estado
- Municipio
- Coordenadas geográficas
- Horarios de atención
- Organizaciones responsables
- Tipos de insumos aceptados

### Escuelas

Contiene información sobre instituciones educativas del país.

Posibles atributos:

- Nombre de la institución
- Nivel educativo
- Dirección
- Estado
- Municipio
- Coordenadas geográficas
- Dependencia (pública o privada)
- Código institucional
- Información de contacto

## Formatos recomendados

Los datos pueden almacenarse en formatos abiertos como:

- CSV
- GeoJSON
- JSON
- XLSX
- Shapefile (SIG)
- GeoPackage (GPKG)

## Objetivos del proyecto

- Facilitar el acceso a datos abiertos de Venezuela.
- Promover el uso de información geoespacial.
- Apoyar iniciativas de investigación y desarrollo.
- Contribuir a proyectos humanitarios y sociales.
- Servir como base para aplicaciones de visualización y análisis territorial.

## Contribuciones

Las contribuciones son bienvenidas.

Antes de enviar cambios:

1. Verifica la calidad de los datos.
2. Incluye las fuentes de información cuando sea posible.
3. Mantén una estructura consistente.
4. Evita duplicados.
5. Documenta cualquier transformación realizada sobre los datos.
